# The deck build tool for Tableturf Battle

This tool is deck build for tableturf battle ([website](https://tableturf-battle.khr32.com/)).

# Development

## Requirement

- node
  see `.nvmrc` for node version

## Install dependencies

```sh
npm install
```

## Local hosting (HMR)

```sh
npm run serve
```

## Build

```sh
npm run build
```
